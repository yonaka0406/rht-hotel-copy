# Accounting Module

This section provides documentation for the Accounting Module, covering financial reporting, ledger management, and data reconciliation.

## Features

- **[Profit & Loss Statement](profit-loss.md)** - Monthly and hotel-specific P&L reports.
- **Ledger Export** - Exporting sales journals in Yayoi format for auditing.
- **[Reconciliation Validation](reconciliation-validation.md)** - Logic and validation procedures for matching PMS sales with actual payments.
- **[Reconciliation Fix Summary](reconciliation-fix-summary.md)** - Technical summary of fixes for sales discrepancy issues.

## Overview

The Accounting Module is designed to streamline hotel financial operations by providing:
1. Automated journal entry generation based on reservation data.
2. Comprehensive P&L reporting with hierarchical management group categorization.
3. Multi-level mapping resolution for plans, addons, and categories to account codes.
4. Robust reconciliation tools to identify and fix discrepancies between sales and payments.
